Model folder

* Contains files for running model.
* Run generate\_mex.m to convert model function to mex file
* Clone https://github.com/gllmflndn/gifti and place as a subfolder in Model folder
* Download S255_high-resolution_group_average_connectome_cortex_nomedial-lh.mat from https://osf.io/xczmp/ and place in Model/Data folder

Experiments folder

* Contains files for producing figures in paper. Named by Figure number.
* Add Model folder and subfolders to path to run files in Experiments folder.
